# 📊 Advanced Sales Analytics Dashboard

An interactive and professional Business Intelligence Dashboard built using Python, Streamlit, Plotly, Pandas, and Machine Learning.

This project analyzes sales data, generates business insights, visualizes trends, performs forecasting, and allows users to download reports in CSV and Excel formats.

---

# 🚀 Features

## 📈 Sales Analytics

* Monthly Sales Trend Analysis
* Sales by Category
* Sales by Region
* Top Product Analysis

## 💹 Profit Analysis

* Profit by Category
* Monthly Profit Trends
* Business Performance Insights

## 🔮 Sales Forecasting

* Machine Learning-based Sales Prediction
* Future Sales Forecast Visualization
* Trend Analysis using Linear Regression

## 🎨 Interactive Dashboard

* Dark Theme UI
* Animated Charts with Plotly
* Sidebar Navigation
* Interactive Filters

## 🔐 Authentication

* Login System
* Session-based User Access

## 📄 Reports

* CSV Report Download
* Excel Report Download

# ⚙ Installation

## 1️⃣ Clone Repository

```bash

```

---

## 2️⃣ Open Project Folder

```bash
cd Advance-Sales-Analytics
```

---

## 3️⃣ Install Dependencies

```bash

```
